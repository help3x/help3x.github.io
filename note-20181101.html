  2. 段落構成は「適切」に

    伝えたい情報の優先度が高い順に書きましょう。 起承転結にこだわる必要はまったくありません。

  3. 一文を短くする

  4. センテンスの長さに変化をつける

    単調なリズムにならないよう意識します。

    たとえば、短いセンテンスが続いたら、次は長いセンテンスにします。 あるいはその逆で、長いセンテンスのあとには短いセンテンスを書きましょう。

  5. 同じ意味のセンテンスを、一つの段落に複数は書かない

    くどい。

  6. 一つの段落やセンテンスの中に、同じ言葉や表現を何度も書かない

  7. 長い段落の後は、短い段落を書く

  8. 接続詞が本当に必要であるかどうかをよく確認する

  9. 専門用語、業界用語、カタカナを極力使わない

  10. 漢字をひらがなにする

    たとえば、断念する→あきらめる、実行する→行うといった要領です。

  11. 一回読んで意味が分からない言葉は使わない

    ひとりよがりな表現ではなく、誰が読んでもスッと理解できる表現を目指しましょう。

  12. 文末に変化をつける

    「～です。～です。～です。」のように同じ文末が繰り返されると、単調で退屈な文章になります。

たとえば、基本料金を23円、ヒーターのワット数を36Wとし、1日に12時間稼働させているとします。

定格電力を元に1ヶ月の電気代を試算してみます。

まずは照明です。

138W（ライトドーム 75W＋紫外線ランプ 13W＋バスキングライト50W） ÷ 1000 × 1日あたり10時間点灯 × 電力量料金 ¥24.87＝1日あたり ¥34.3206

1ヶ月（30日換算）にすると ¥1029.618 ≒ ¥1030 となります。

次に水中ヒーターです。

55W ÷ 1000 × 1日あたり24時間通電 × 電力量料金 ¥24.87 ＝ 1日あたり ¥32.8284

1ヶ月（30日換算）にすると ¥984.852 ≒ ¥985 となります。

照明とヒーターを合わせて、1ヶ月あたり ¥2,015 が想定される電気代になります。

と言っても、水中ヒーターが24時間つきっ放しになることは、おそらくないと思いますので、これよりは安くなるはずです。

GitHub Pages のポイント

概要

  • GitHub Pages とは静的サイトのホスティングサービスのこと。
  • Jekyll テーマセレクタを使ったサイトの作成・公開ができる。
  • HTTPS通信をサポートしている。
  • 非公開リポジトリであっても、GitHub Pages のサイトはインターネット上で公開される。
    機密データは除いて公開しなければならない。

使用制限

  • リポジトリの推奨制限は1GB。これを超えると GitHub サポートから、リポジトリのサイズを縮小するようメールが送られることがある。
  • 公開されている GitHub Pages サイトは、1GB以下にしなければならない。
  • 1ヶ月あたり100GBの帯域幅制限がある。
    …1ヶ月に転送できるデータ量は100GBまで、という意味。
  • 1時間あたり10ビルドの制限がある。

使用制限を超えたとき

制限を超えたサイトに対して、サービスを提供できなくなる。

必要に応じて、異なるホスティングサービスへの移行をお願いする。

キーワードの種類、目的、特徴

「行動」キーワード
目的
「○○したい」といったように、行動をするため。
特徴
このキーワードで上位表示できれば、それだけで成果が発生する。
ただ、上位表示は難しいため、最下層の記事からアクセスを集めて、誘導、上位表示を目指したい。
「商標・ブランド」キーワード
目的
目的のブログ、ウェブサイトへのアクセスをすること。
特徴
AmazonやYouTubeなどのように、目的とするサイトに訪れるためのSEOキーワード。
最終的に自身の「商標・ブランド」キーワードで1位の獲得を目指す。
「情報収集」キーワード　* 最重要！
目的
疑問や悩みを解決する答えを知ること。
特徴
最も検索されるキーワードであり、ユーザーの大半はここから入ってくる。
訪れた人の悩みを解決する記事を多数用意しておくことで、あなたのことを「その分野の専門家」として見てくれるようになる。
集客に最適なSEOキーワードであるといえる。
「行動」＆「情報収集」キーワード
目的
情報収集した上で、行動しようとしている。
特徴
悩みを解決する手段や方法を知っているが、最後に「どれ、どこが良いのか」を情報収集してから行動（購入や申し込み）をしようとしている。
アフィリエイトサイトでは、このSEOキーワード群を徹底的に網羅することで、少ないアクセスでも高い契約率を実現できる。
「行動」＆「商標・ブランド」キーワード
目的
行動を起こすために、特定のブログ、ウェブサイトへいこうとしている。
特徴
たとえば、「庭の工事の見積もり（行動）」を行うために「会社名（商標・ブランド）」を検索するパターン。
「商標・ブランド」＆「情報収集」キーワード
目的
情報収集のために、特定のブログ、ウェブサイトへいこうとしている。
特徴
自身のブログやサイトが有名になると、このキーワードで上位表示されるようになる。
会社やブログ、サイトの名前を含めて検索されるため、ファンからのアクセスを見込める。
