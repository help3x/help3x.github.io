  • 第三章 起承転結・四百字基本形を利用した意見文の取り立て指導

    起承転結を会得するには、いろいろな方法があります。 そのなかで、四百字原稿用紙を起・承・転・結に分けて書く練習が、最も効果的だと思います。 作家の久美紗織の本で知って以来、私も文章を教える基本にしています。 最初の三行が起、次の九行が承、これ …

  • 起承転結は四行詩の定型。文章に向いてない理由… - 文章力をつける

    小論文は結論から書くものですし、エッセイも起承転結の「承」で間延びし、「転」の位置が不自由なのです。 本講座でおすすめ … が致命的な欠陥です。 随筆や散文を起承転結の形式で書こうとするなら、四つの段落で収まる200字～400字くらいが適量でしょう。

path.dirname(path)

The path.dirname() method returns the directory name of a path, similar to the Unix dirname command. Trailing directory separators are ignored, see path.sep .

path.dirname('/foo/bar/baz/asdf/quux');
// Returns: '/foo/bar/baz/asdf'
