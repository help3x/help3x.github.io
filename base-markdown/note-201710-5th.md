### ガラスとアクリルの特徴

ガラスは傷が付きにくく、透明度の変化が少ない。しかし、重い。割れやすい。割れたときに小さい破片になりやすい。

アクリルは軽くて割れにくい。割れたとしても小さな破片になりにくい。しかし、傷が付きやすく、ゆがみなどの経年劣化による変形が起こりやすい。


アクリル製にしたい。
万が一、地震で落ちたときに、ガラス製だと割れて玄関中に破片が飛び散る可能性がある。
しかし、製品の選択肢が少ない。

黙ってセット商品で始めるのもありか。


電気を使いたくない。

玄関にコンセントはあるが、延長コードで引っ張ってくると見栄えが悪い。

フィルターを動かすには電気が必要。
電池式のエアーポンプは長時間稼働を目的に作られていないので、これで代用するのは無理。


毎日、コップ２～３杯分の水を交換する。
前夜に容器に交換用の水を準備しておく。
翌朝に糞を取り除いてから水を交換する。
手間はかかるが電気を使わないのであればこうするしかない。

## コンセントの容量

定格容量ギリギリ、あるいは定格容量をオーバーする接続は危険です。

たとえば、定格容量1500Wに対してトースター1200Wと電気ポット300Wで合計1500W。
定格容量1500Wに対してホットカーペット1100W、こたつ500Wの合計1600W。
こういった接続は危険です。

電化製品の消費電力と、コンセントやテーブルタップの最大容量を確認して使う。

## ボディメイク・スクワット

### やり方

#. 両足を肩幅ぐらいに開いて立ちます。

#. 背中をまっすぐに伸ばして、顔、おへそ、つま先、ひざは、まっすぐ前を向いた状態にします。

#. 両腕は頭の後ろに組む、または胸の前で交差させるなど、自分が動きやすい姿勢をとります。

#. 鼻から息を吸いながら、1から8までカウントしつつゆっくりと腰を落とし、8で太ももと床が平行になるようにします。ひざが内側を向かないように注意し、かかとは浮かないように床にしっかりとつけてください。横から見たときに、お尻が後ろに突き出たような姿勢になります。

#. 息を吐きながら、元の姿勢に戻ります。元の姿勢に戻すときは、1拍で戻すぐらいの速さで、スッと腰を伸ばします。


### Point!

* 腰を落とすときに『1、2……7、8』とカウントしながらゆっくりと動くことです。これは筋肉の少ない人が筋肉量を増やすために行う運動で、ゆっくりと腰を落とすことで普段は鍛えていない筋肉に働きかけていきます。

* 実施する回数は、自分の限界＋1回。限界を少しだけ超えると、筋肉が反応してくれるようになる。

* 慣れてきたら1セットあたりの回数を増やしてみる。あるいはペットボトルなどを持って重量をかけて行う。












## font-size: 62.5%の意味

font-sizeが未指定だとデフォルトのフォントサイズが 16px になる。  
これだと相対指定でフォントサイズを変更するときに計算しにくい。  
なのでデフォルトのフォントサイズが 10px になるよう 62.5% を設定する。  
（16px × 62.5% ＝ 10px ってことです）  
その上で、bodyのフォントサイズを 1.6倍、つまり 16px になるように指定する。

```
/* まずはhtml要素のフォントサイズを 10px に設定する */
/* （デフォルトのフォントサイズ 16px × 62.5% ＝ 10px） */
html {
  font-size: 62.5%;  
}

/* 続いてbody要素のフォントサイズを 16px に設定する */
/* （html要素のフォントサイズ 10px × 1.6 ＝ 16px） */
body {
  font-size: 1.6rem;
}
```
