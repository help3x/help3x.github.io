## Galaxy Feel SC-04J

### 特長



幅約67mm、薄さ約8.3mmのスリム設計で、手にしっとりとなじむ、すぐれたグリップ感を実現。カメラ部の突出もなくしフラットに仕上げました。バッグやポケットに入れてもかさばりません。 



硬質なメタル素材と2.5Dガラスを使用し、シンプルかつ上質な仕上がりを実現しました。ガラスは両面ともに強度の強いGorilla® Glass 4を使用し、傷や汚れから守ります。また、有機ELディスプレイを採用しているので、約4.7インチとコンパクト設計でありながら、屋外でも見やすく、色鮮やかで美しい画像をお楽しみいただけます。 



雨の日や水回りでも
安心の防水・防塵対応

キャップレス防水・防塵に対応（IPX5/8、IP6X)。水回りのさまざまなシーンでも安心して使用できます。 

* ご使用の際は、SIMトレーを確実に取り付けてください。
* 本体内に水分が残っている状態で充電はできません。

### VB6のサブメニューの制約

すべてのサブメニューを不可視化することはできません。  
1つ以上のサブメニュー項目を可視化にする必要があります。

メニューエディタですべて不可視化にすると、エラーメッセージが表示されました。  
コード上ですべて不可視化にすると、最後のサブメニューを不可視化にしたとき、実行時エラー "387" が発生しました。

残念。  
いったん不可視化で初期化しておいて、必要なサブメニューだけ表示という処理はできないようです。

### 構想

次のURLになるようにする。  
"sample-post" 部分はページ毎に指定する。

　→ http://hoge.com/sample-post/index.html

Markdownファイルをベースにし、Pandocを介して、HTMLを生成します。  
ファイルのYAMLメタデータブロックに生成に必要な値を入力してください。

Pandocに渡すMarkdown
:   inputフォルダ配下のMarkdown (*.md) ファイル。

Pandocが生成するHTML
:   outputフォルダ配下に入力元のMarkdown (*.md) ファイルの名前でフォルダを作る。<br>
    その中に`index.html`と言う名前でファイルを出力する。  
    Pandocの`-o/--output`オプションに指定する。


#### フォルダ構成

inputに元のファイルを保存。  
生成した結果はoutputに吐く。

フォルダのパスは当分決め打ち。

生成するときはHomeフォルダを基点にする。

```
(Home)/
+-- input
|   +-- images
|   +-- template
|   |   +-- amp.html ... AMPに対応したHTMLテンプレート
|   +-- css
|   |   +-- style.css ... amp-customに入れるCSS
|   +-- script
|   +-- sample-post-1.md
|   +-- sample-post-2.md
|   :
|   +-- index.html
|
+-- output
    +-- images
    +-- sample-post-1
    |   +-- index.html
    +-- sample-post-2
    |   +-- index.html
    :
    +-- index.html
```


テンプレート変数

pagetitle
:   ページタイトルです。

title-prefix
:   ページタイトルの前に挿入する文字列。&#x2d; で繋ぎます。

title-suffix
:   ページタイトルの後に追加する文字列。&#x7c; で繋ぎます。

こんな感じ  
ページプレフィクス &#x2d; ページタイトル &#x7c; ページサフィックス

custom-css
:   スタイル定義です。
