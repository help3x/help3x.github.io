## 日本語Webフォントは重い？

無料の日本語Webフォント「Noto Sans Japanese」というものがある。  
ファイルサイズが大きく、1フォントあたり1.5MBほどの容量があった。

容量削減の方法に、フォントのサブセット化がある。

サブセット化とは、フォントに含まれている文字の中から必要なものだけを抜き出すことを言う。  
日本語フォントであれば、「ひらがな＋カタカナ＋第一水準漢字（2965字）＋記号」あたりを抜き出してサブセット化すればだいぶ軽量化できる。

軽量化の効果として、サブセット化する前は1.5MBだったのが、した後は500KBぐらいになった模様。

必要なもの

* 日本語フォント
    * 元にするフォントファイル。
* サブセットフォントメーカー
    * フォントのサブセット化に使う。
* woffコンバーター
    * サブセット化したフォントファイルを、Webフォントとして使える形式に変換する。

## 読書メモ

関係を意識した短作文を積み重ねることが第一歩となる。

#. 「言いかえる力」　同等関係（抽象・具体の関係）を整理する力  
  「Ａ。つまり、Ｂ」「Ｂ。たとえば、Ａ」

#. 「くらべる力」　対比関係を整理する力  
  「Ａ。しかし、Ｂ」「Ａ。一方、Ｂ」

#. 「たどる力」　因果関係を整理する力。  
  「Ａ。だから、Ｂ」「Ｂ。なぜなら、Ａ」


### 言いかえる力: 同等関係

抽象化
　「まとめながら」言いかえること。

　みかん、りんご、ばなな。つまり果物。

具体化
　「分けながら」「詳しくしながら」言いかえること。

　果物。たとえば、みかん、りんご、ばなな。



### くらべる力: 対比関係

「それに対して」「一方・しかし」でつなぐ。

「正反対」

白　　　　　　　　黒  
明るい　＜－＞　　暗い  
心　　　　　　　　物

* 白は明るい色。それに対して黒は暗い色。
* 豊かになったのは、心ではなく物である。


「ワンセット」　正反対の意味をもつものではないが、一つのセットとして成立する関係。

熱い　　　　　ぬるい  
バス　＜－＞　タクシー  
日本　　　　　アメリカ



### たどる力: 因果関係

原因: 忘れ物をした

↓　だから　　　↑　なぜなら

結果: 叱られた。


以下のように連鎖することもある。

原因⇔結果  
　　　原因⇔結果  
　　　　　　原因⇔結果


* 忘れ物をした。だから、叱られた。
* 叱られた。なぜなら、忘れ物をしたからだ。

原因と結果の関係を「因果関係」と言う。  
因果関係をたどって文にすることもある。  

以下はその例。ややくどいが。  
　寒い。だから、厚着をしている。だから、暖房が逆に熱い。だから、設定温度を下げよう。

どの程度細かくたどるべきかは「相手の常識」に依存する。

## ImageMagick

### テキスト画像を作る

`label`で指定する方法。  
指定したフォントサイズでテキストを流し込む。

```
C:\Work>convert.exe -pointsize 24 label:image-magick.com img01.jpg
```

`-pointsize`を指定せず、`-size`を指定することで、そのサイズに合わせたフォントサイズでテキストが流し込まれる。

```
C:\Work>convert.exe -size 640x label:image-magick.com img01-1.jpg
```

`caption`で指定する方法。  
指定したサイズに合わせてテキストを流し込む。

```
C:\Work>convert.exe -size 199x28 caption:image-magick.com img02.jpg
```

`-pointsize`を指定しない、または`-size`に幅と高さの両方を指定することで、自動的にフォントサイズを調整してくれる。

```
C:\Work>convert.exe -size 320x240 caption:"ImageMagick image-magick.com" img02-1.jpg
```
