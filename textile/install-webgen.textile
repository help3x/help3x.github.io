一時的にhttpを追加する。

<pre>
C:\Users\hoge>gem sources --add http://rubygems.org/
https://rubygems.org is recommended for security over http://rubygems.org/

Do you want to add this insecure source? [yn]  y
http://rubygems.org/ added to sources
</pre>

終わったらhttpを外す。

<pre>
C:\Users\hoge>gem sources
*** CURRENT SOURCES ***

https://rubygems.org/
http://rubygems.org/
</pre>

<pre>
C:\Users\hoge>gem sources --remove http://rubygems.org/
http://rubygems.org/ removed from sources
</pre>

<pre>
C:\Users\hoge>gem sources
*** CURRENT SOURCES ***

https://rubygems.org/
</pre>
